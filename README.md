# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript with type-aware lint rules enabled. Check out the [TS template](https://github.com/vitejs/vite/tree/main/packages/create-vite/template-react-ts) for information on how to integrate TypeScript and [`typescript-eslint`](https://typescript-eslint.io) in your project.

                                                        🚀 TECHNICAL IMPLEMENTATION DETAILS

1. Header Section (TopBar & Navbar)

Tech Used: Sticky Positioning aur Bootstrap Navbar ka use kiya hai responsive navigation ke liye.
Logic: Top-bar ko info ke liye aur main navbar ko smooth navigation links ke liye align kiya hai.

2. Hero Section (Video Banner)

Tech Used: HTML5 <video> tag with autoPlay, muted, and loop attributes.
Logic: object-fit: cover property use ki hai taaki video bina stretch hue poori width cover kare.

3. Thumbnail Section (Match Schedule)

Tech Used: Flexbox (d-flex) aur Overflow-X: auto.
Logic: Custom CSS Webkit-Scrollbars banaye hain taaki horizontal scroll professional aur "IPL App" jaisa dikhe.

4. Points Table (Team position)

Tech Used: CSS Grid and Clip-path.
Logic: "Full Points Table" button ke liye Polygon Clip-path use kiya hai taaki wo slanting/tilted look aaye jo official site par hai.

5. Sponsors Section

Tech Used: Bootstrap Grid System (Rows & Cols).
Logic: Multi-level alignment use ki hai jisme Title Sponsor (Tata) ko highlight kiya hai aur baki partners ko symmetric grid mein rakha hai.

6. Footer Section

Tech Used: Semantic HTML5 and Background Overlays.
Logic: 4-column layout ke saath Spiral Background Image integrate ki hai aur bottom mein ek alag Copyright bar banayi hai.
